Previsão de Notas - Estudo de machine learning
ⓘ Como funciona
Essa aplicação utiliza um conceito de Machine Learning - a Regressão Linear, para fazer previsões baseadas em dados já existentes. Faça o upload de um arquivo .csv e experimente!
ⓘ Arquivos aceitos
Essa aplicação aceita arquivos com duas colunas: [Semestre] e [Nota], e seus respectivos valores.
